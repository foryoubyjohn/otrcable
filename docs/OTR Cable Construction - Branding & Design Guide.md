## Website Layout Guidelines

### Header
- Logo on the left
- Navigation menu (horizontal)
- Contact button (CTA) on the right
- Blue gradient background

### Hero Section
- Large heading with company value proposition
- Subheading describing services
- Two CTA buttons (Get Quote, Learn More)
- Background image with blue overlay

### Service Cards
- Grid layout (3 columns on desktop, 1 on mobile)
- White background with subtle shadow
- Service icon (orange)
- Service title (navy blue)
- Brief description (dark gray)
- "Learn More" link (primary blue)

### Footer
- Dark navy background
- White text
- Company information, services, contact details
- Social media links
- Copyright notice

## Content Tone & Voice

### Brand Personality
- **Professional**: Expert knowledge and reliable service
- **Innovative**: Cutting-edge technology solutions
- **Approachable**: Easy to work with and understand
- **Trustworthy**: Dependable and secure installations

### Writing Style
- Clear and concise
- Technical accuracy without jargon
- Solution-focused
- Customer-centric
- Professional but friendly

## Responsive Design

### Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px+

### Mobile Considerations
- Touch-friendly buttons (44px minimum)
- Simplified navigation (hamburger menu)
- Stacked layouts
- Optimized images
- Fast loading times

## Accessibility

### Standards
- WCAG 2.1 AA compliance
- Color contrast ratio 4.5:1 minimum
- Keyboard navigation support
- Screen reader compatibility
- Alt text for all images

### Implementation
- Semantic HTML structure
- ARIA labels where needed
- Focus indicators
- Skip navigation links
- Descriptive link text
